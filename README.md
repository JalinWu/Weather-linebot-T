# Weather-linebot-S

[For Student]

It's a teaching material for practicing linebot, which is based on Node.js, Express and linebot. Have fun and enjoy it.

### Version: 1.0.0

### Usage

```js
// .env, fill in your personal data
LINE_CHANNEL_ID=""
LIEN_CHANNEL_SECRET=""
LINE_CHANNEL_ACCESS_TOKEN=""
```

```sh
$ npm install
```

```sh
$ npm run dev
```

```sh
# mac
$ ./ngrok http 3000

# windows
$ ngrok-win http 3000
```
